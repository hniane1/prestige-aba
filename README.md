A professional, modern website for Prestige ABA Therapy - an Applied Behavior Analysis therapy center helping children with autism and developmental disabilities.

**Website:** https://hniane.com/prestige-aba/

## 🔧 Customization Guide

### Change Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-blue: #1e5a96;
    --secondary-blue: #4a90e2;
    --orange: #f5a623;
}
```

### Update Content
All content is in `index.html`:
- Services descriptions
- Team member info
- Contact details
- About section text

### Modify Logo
Replace `logo.jpg` with your own logo file
